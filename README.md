# Laravue Fullstack

Laravue-Fullstack is a complete solution for API and frontend which comprises of Laravel 6.2/OpenAPI 3.0/oAuth 2.0/GuzzleHTTP as the backend API and Laravel 6.2/Vue JS Dashboard as the frontend.
